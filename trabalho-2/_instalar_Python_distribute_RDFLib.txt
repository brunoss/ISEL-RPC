- Windows 10 | 32 bit

Informação sobre o RDFLib (excerto do site; em mar.2018):
"RDFLib is a Python library for working with RDF, a simple yet powerful language for representing information.

The library contains parsers and serializers for RDF/XML, N3, NTriples, Turtle, TriX, RDFa and Microdata. The library presents a Graph interface which can be backed by any one of a number of Store implementations. The core rdflib includes store implementations for in memory storage, persistent storage on top of the Berkeley DB, and a wrapper for remote SPARQL endpoints.

A SPARQL 1.1 engine is also included."

** Dicas para detecção e resolução de alguns problemas relacionados com a execução do IDLE**
_______________________________________________________________________________________________
O ambiente IDLE inclui um editor, um interpretador e um "debugger" muito simples para o Python

Caso tenha dificuldade em iniciar o ambiente IDLE
(em geral resulta de conflito com outra(s) instalações Python)
fazer, em linha de comando e assumindo que o Python está instalado em :

C:\...\Python36\pythonw C:\...\Python36\Lib\idlelib\idle.py
(isto deve funcionar mesmo que tenha conflito com outras instalações Python)

Caso o IDLE execute correctamente uma solução será construir um ficheiro:
idle.bat (e colocá-lo na pasta "C:\...\Python36\") de modo a que fique com código equivalente à expressão acima

Caso o IDLE não execute correctamente , tentar
- iniciar o IDLE como "Administrator"
- iniciar o IDLE  depois de desligar a firewall do Windows (mais perigoso)

:: em versão anterior do Python era necessário instalar
- Instalar "setup_tools"
- que contém ferramenta para instalar distribuições de outros pacotes
>> a versão Python 3.4.3 (e posteriores) já inclui o "setup_tools"
>> assim não é necessário fazer a instalação separada do "setup_tools"
>> no entanto (para efeitos de memória) fica abaixo a descrição da última instalação efetuada
_______________________________________________________________________________________________
1. descomprimir "distribute-0.6.49.tar" (na pasta "distribuicao_01")
   OU
   descarregar "distribute 0.6.49" de:
    "https://pypi.python.org/pypi/distribute/0.6.49"
   e descomprimir
2. abrir uma "Command Prompt" (janela DOS)
3. mover para a pasta (...)\distribute-0.6.49
4. executar:
   python distribute_setup.py 

Depois desta instalação ver a pasta "...\Python33\Lib\site-packages" que
deverá agora deve conter "distribute-0.6.49-py3.3.egg" onde está o "easy_install.py"
